# Drop your 360° photos here

Put one equirectangular (2:1) JPG per scene in this folder, named to match the
scenes in [`../../js/data/comaruga.js`](../../js/data/comaruga.js):

```
start.jpg  wp1.jpg  wp2.jpg  wp3.jpg  wp4.jpg  station.jpg
```

Then set each scene's `image` field to the filename. Until you do, the game
auto-generates placeholder panoramas, so it still runs.

Full instructions: [`docs/CAPTURE_GUIDE.md`](../../../docs/CAPTURE_GUIDE.md).
